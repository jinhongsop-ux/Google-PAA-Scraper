================================================================================
                    PAA Scraper - 使用说明
================================================================================

📋 简介
------
PAA Scraper 是一个自动化工具，用于从 Google 搜索中提取 "People Also Ask" (PAA) 
问题和相关搜索词。支持断点续传和智能去重功能。


🚀 快速开始
----------

1. 首次使用前，请编辑 config.json 文件配置您的关键词

2. 双击运行 PAA_Scraper.exe

3. 程序会自动：
   - 创建 results 文件夹
   - 为每个关键词生成独立的 Excel 文件
   - 实时保存抓取结果


⚙️ 配置文件说明 (config.json)
----------------------------

打开 config.json，您会看到以下内容：

{
    "keywords": [
        "python automation",
        "AI technology"
    ],
    "max_depth": 3,
    "headless": false
}

配置项说明：

• keywords (必填)
  要抓取的关键词列表，可以添加多个关键词
  示例：["Pickleball Paddles", "SEO Tools", "Python Programming"]

• max_depth (可选，默认: 3)
  PAA 问题展开的深度级别
  数值越大，抓取的问题越多，但耗时也越长
  建议范围：2-5

• headless (可选，默认: false)
  是否使用无头模式（不显示浏览器窗口）
  true = 后台运行，不显示浏览器
  false = 显示浏览器窗口，可以看到抓取过程


📂 查看结果
----------

运行完成后，打开 results 文件夹，您会看到：

results/
├── python_automation.xlsx
├── AI_technology.xlsx
└── Pickleball_Paddles.xlsx

每个关键词对应一个 Excel 文件，文件名会自动：
- 将空格转换为下划线
- 过滤非法字符（如 <>:"/\|?*）


🔄 断点续传功能
--------------

如果程序中途崩溃或您手动停止：

1. 已抓取的数据已安全保存在 results 文件夹中

2. 重新运行程序时，会自动：
   - 加载已有数据
   - 跳过重复的问题
   - 只抓取新发现的问题

控制台会显示：
📚 已加载关键词 [XXX] 的历史数据 15 条，将自动跳过重复...


📊 Excel 文件结构
----------------

每个 Excel 文件包含以下列：

• Original Keyword - 原始关键词
• Type - 类型（PAA 或 Related Search）
• Question/Term - 问题或搜索词
• Snippet - 答案摘要
• Source Link - 来源链接
• Discovery Level - 发现层级


⚠️ 注意事项
----------

1. 首次运行时，程序会自动下载 Chrome 浏览器驱动
   请确保您的电脑已安装 Google Chrome 浏览器

2. 如果遇到 CAPTCHA（验证码），程序会暂停
   请在浏览器中手动完成验证，然后按 Enter 继续

3. 抓取速度较慢是正常的，这是为了模拟人类行为，避免被 Google 封禁

4. 建议不要同时抓取过多关键词，可以分批进行


🛠️ 常见问题
-----------

Q: 程序运行时提示找不到 Chrome 浏览器？
A: 请先安装 Google Chrome 浏览器
   下载地址：https://www.google.com/chrome/

Q: 如何修改关键词？
A: 用记事本打开 config.json，修改 keywords 列表，保存后重新运行程序

Q: 结果文件在哪里？
A: 在程序所在目录的 results 文件夹中

Q: 如何清空历史数据重新抓取？
A: 删除 results 文件夹中对应的 Excel 文件即可

Q: 程序运行很慢？
A: 这是正常的，为了避免被 Google 检测，程序会模拟人类行为
   您可以减少 max_depth 的值来加快速度


📧 技术支持
----------

如有问题，请检查：
1. 是否已安装 Google Chrome 浏览器
2. config.json 格式是否正确（注意逗号和引号）
3. 网络连接是否正常
